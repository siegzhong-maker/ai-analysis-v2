# 快速启动指南

## 给研发的演示准备

### 1. 安装依赖（首次运行）

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

服务器会在 `http://localhost:5173` 启动

### 3. 演示要点

#### 最近项目模块
- 首页"最近项目"标签页显示4个项目：
  - 2个足球项目（基础集锦 + 高阶分析）
  - 2个篮球项目（基础集锦 + 高阶分析）

#### 基础集锦视图（点击基础集锦项目）
- **智能集锦标签页**：
  - 足球：显示进球、角球、定位球、点球等事件
  - 篮球：显示3分、2分、罚球等得分事件
  - 支持按队伍和事件类型筛选
  - 支持片段选择和批量操作

- **基础数据标签页**：
  - 足球：显示进球等核心数据
  - 篮球：显示总出手、命中等核心数据
  - 关键事件时间轴
  - 升级提示横幅

#### 高阶分析视图（点击高阶分析项目）
- **数据概览标签页**：
  - 完整的数据对比表格
  - 关键事件时间轴
  - 支持按事件类型筛选

- **高阶分析标签页**：
  - 足球：跑动热力图 + 体能数据
  - 篮球：投篮热点分布 + 球员效率榜

### 4. 关键功能演示路径

1. **足球基础集锦**：
   - 首页 → 点击"周日联赛：进球集锦"
   - 查看智能集锦标签页（足球事件筛选）
   - 切换到基础数据标签页（进球数据）

2. **足球高阶分析**：
   - 首页 → 点击"半决赛：战术分析"
   - 查看数据概览标签页（完整数据对比）
   - 切换到高阶分析标签页（跑动热力图 + 体能数据）

3. **篮球基础集锦**：
   - 首页 → 点击"周末友谊赛集锦"
   - 查看智能集锦标签页（篮球得分事件筛选）
   - 切换到基础数据标签页（出手和命中数据）

4. **篮球高阶分析**：
   - 首页 → 点击"决赛：数据统计报告"
   - 查看数据概览标签页（完整数据对比）
   - 切换到高阶分析标签页（投篮热力图 + 球员效率榜）

### 5. 注意事项

- 所有数据为模拟数据，用于演示UI和交互流程
- 需要网络连接以加载Tailwind CSS CDN
- 建议使用Chrome浏览器以获得最佳体验
- 如果遇到类型错误，运行 `npm install` 后重启开发服务器

### 6. 技术栈说明

- **React 18** - UI框架
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **Tailwind CSS** - 样式框架（CDN）
- **Lucide React** - 图标库

## 常见问题

**Q: 为什么有这么多TypeScript错误？**  
A: 这些错误是因为还没有安装依赖。运行 `npm install` 后就会消失。

**Q: 如何修改数据？**  
A: 在 `App.tsx` 文件中找到对应的常量（如 `HISTORY_TASKS`、`AI_CLIPS_ADVANCED` 等）进行修改。

**Q: 如何添加新的运动类型？**  
A: 在 `App.tsx` 中添加新的运动数据，并在相关组件中添加对应的判断逻辑。
