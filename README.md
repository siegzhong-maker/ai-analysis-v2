# AI Analysis High-Fidelity Prototype

这是一个高保真原型，用于演示 AI 分析功能：提供**集锦**与**全场数据分析**两种结果形态，支持足球和篮球。产品采用**单一会员订阅**：未订阅可体验智能集锦；订阅后解锁数据分析、高级成片模板、无水印分享等全部付费能力（无 Basic / Pro 双档）。

## 功能特性

### 最近项目模块
- 2个足球项目（集锦报告 + 数据报告）
- 2个篮球项目（集锦报告 + 数据报告）

### 集锦结果视图 (HighlightResultScreen)
- **智能集锦标签页**：
  - 足球：显示进球、角球、定位球、点球等事件
  - 篮球：显示3分、2分、罚球、篮板、抢断、助攻等事件
  - 支持按队伍和事件类型筛选
  - 支持片段选择和批量操作

- **基础数据标签页**：
  - **个人数据看板**（优先展示）：
    - 篮球：得分、篮板、助攻、抢断、效率值等个人统计
    - 足球：进球、角球、定位球、点球等个人统计
  - **团队对比数据**（折叠展示）：
    - 篮球：总得分、篮板、抢断等团队对比
    - 足球：进球等核心数据对比
  - **热力图**（免费功能）：
    - 篮球：出手热力图
    - 足球：跑动热力图
  - 升级提示横幅

### 全场数据分析视图 (AnalysisResultScreen)
- **数据概览标签页**：
  - 完整的数据对比表格（包含篮板、抢断等数据）
  - 关键事件时间轴
  - 支持按事件类型筛选

- **高阶分析标签页**：
  - 足球：跑动热力图 + 体能数据
  - 篮球：投篮热点分布 + 球员效率榜

## 核心数据

- `HISTORY_TASKS`: 最近项目列表（4个项目）
- `AI_CLIPS_ADVANCED`: AI识别的精彩片段
- `TEAM_MATCH_STATS`: 篮球比赛统计数据
- `SOCCER_MATCH_STATS`: 足球比赛统计数据
- `PLAYER_STATS_BASKETBALL`: 篮球球员数据
- `PLAYER_STATS_SOCCER`: 足球球员数据

## 功能划分

### 未订阅（原型演示）
- ✅ 进入智能集锦流程并查看集锦结果
- ✅ 片段选择与批量操作、基础数据与热力图等集锦页能力（以原型数据演示）

### 会员订阅（单一档位，权益一致）
- ✅ 全场数据分析入口与完整数据报告视图
- ✅ 全部成片模板（含叙事/品牌/影院级等档位）
- ✅ 高级视频编辑（多片段合并、自定义时长、特效）
- ✅ 数据导出（PDF 报告、Excel 表格）与更深复盘视图
- ✅ 分享增强（会员可无水印导出等，以原型交互演示）
- ✅ 跟练视频与训练库相关权益（文案与流程以原型演示）

## 使用说明

1. 首页显示4个最近项目（2个足球 + 2个篮球）
2. 点击集锦类项目 → 进入集锦结果视图（智能集锦 + 数据摘要等）
3. 点击数据分析类项目 → 进入全场数据分析视图（数据概览 + 复盘）
4. 在结果页面可以切换标签页查看不同内容
5. 支持在两种结果形态间跳转（以原型导航为准）

## 在线部署

### 使用Vercel部署（推荐）

1. 访问 [vercel.com](https://vercel.com) 并使用GitHub账号登录
2. 点击 "Add New Project"
3. 导入仓库 `siegzhong-maker/ai-analysis-v2`
4. Vercel会自动检测Vite项目配置
5. 点击 "Deploy" 开始部署
6. 部署完成后会获得一个URL（如：`https://ai-analysis-v2.vercel.app`）

### 在线访问

部署完成后，可以通过Vercel提供的URL访问应用，分享给其他研发同学查看。
